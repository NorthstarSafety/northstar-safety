# Northstar Safety Phase 7 Memo

Historical note: this memo reflects an earlier pilot structure. The current live founding pilot is $300 one-time for 30 days and the first 10 priority SKUs.

## What changed

Phase 7 moved Northstar from "ready to sell" to "ready to run founder-led outbound."

The work focused on practical selling assets instead of new strategy:

- prioritized lead list
- outreach cadence
- founder demo checklist
- objection handling
- pipeline tracker

## Offer changes

The pilot offer is now tighter:

- 45-day pilot term
- one Shopify store
- first 25 priority SKUs actively configured during onboarding
- founder-led kickoff plus one follow-up working session

That makes the offer easier to explain and protects the support ceiling.

## Lead selection logic

The best early targets are brands with:

- real child-product compliance exposure
- public safety or warranty language
- visible Shopify storefront signals
- founder or small-team accessibility

## Selling posture

The goal is not broad top-of-funnel volume.

The goal is:

- 5 high-quality first touches per day
- 5 smart follow-ups per day
- fast demos on a live Northstar workspace
- clear next steps within 48 hours after each call

## Remaining deployment gap

Northstar still does not have a persistent hosted environment because no cloud host credentials were available in this workspace. The local demo path is polished, and deployment handoff is clean, but a stable shareable URL still requires founder infrastructure access.
