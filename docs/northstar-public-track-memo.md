# Northstar Public Presence Track Memo

Date: 2026-03-23

## What changed

Northstar now has a real public-facing company layer instead of just an internal pilot workspace.

That includes:

- a public website at `/`
- pricing, Shopify, security, support, privacy, and terms pages
- a working public contact form
- operator UX copy that is easier to understand
- deployment assets aligned to the real Northstar app
- Shopify review-readiness scaffolding and docs

## Product clarity improvements

The operator workspace now explains itself better:

- dashboard copy is clearer
- catalog coverage language is simpler
- proof queue language is simpler
- alert review language is less jargon-heavy
- investigation pages better explain what an operator should do next

## Trust improvements

- public company pages are now present
- support and contact routes are explicit
- privacy and terms pages are public
- the workspace can stay protected while the public site remains open

## What still blocks full public readiness

- no real public deployment credentials
- no branded support inbox or scheduling link
- no OAuth install flow yet
- no Shopify Billing API / Managed Pricing flow yet
- no final App Store listing assets or reviewer account package yet

## Bottom line

Northstar now has a credible public presence and a much cleaner path to public Shopify readiness, but it is still correctly positioned as a private pilot product until OAuth, billing, and final listing assets are built.
